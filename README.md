# Freelancer-Hub-Global-Income-Skill-Analysis-Dashboard-2025-
This project analyzes the global freelance market in 2025 using Microsoft Excel. The dashboard highlights freelancer income trends, regional pay differences, platform usage, skill demand, and niche opportunities across the freelance economy.

The goal of this project is to transform raw freelancer data into meaningful insights that help freelancers, recruiters, and businesses make smarter decisions.

## Project Objectives

This analysis was designed to answer the following questions:

- Which freelance skills generate the highest income?
- Which regions offer the best hourly rates?
- Which freelance platforms are most used?
- How does experience level affect earnings?
- What niche skills are currently profitable?
- Where are freelancers located globally?

## Dataset Information

The dataset contains global freelancer records and includes the following fields:

- Freelancer ID
- Country
- Region
- Hourly Rate (USD)
- Annual Income (USD)
- Primary Skill
- Niche Skill
- Platform Used
- Experience Level
- Education Level
